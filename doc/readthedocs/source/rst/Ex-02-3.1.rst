Find Positive Backbone
========


An example of finding the backbone curve for data from a siesmic damper.
OpenSees is used to generate input data for the damper.\
The following data is used in the example: 

#. Displacement data  :download:`here </examples/03 Find Backbone of Experimental Data/3.1/Disp.out>`.
#. Reaction force data :download:`here </examples/03 Find Backbone of Experimental Data/3.1/RFrc.out>`.
